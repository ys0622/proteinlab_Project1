## 추가 방법

1. 위 파일명으로 이미지를 준비 (흰색 배경, 정면샷)
2. `Bar image` 폴더에 저장
3. 아래 명령 실행:
   ```bash
   node scripts/sync-bar-images-from-folder.mjs
   ```

> 위 22개 제품은 이미 sync 스크립트에 매핑되어 있습니다. 이미지를 `Bar image` 폴더에 넣고 스크립트만 실행하면 됩니다.
